# LeAddAds - WordPress 内容广告插件

LeAddAds 是一个简单而强大的 WordPress 插件，它允许你在文章内容的指定段落后插入广告代码。

## 功能特点

- 可以轻松开启/关闭广告插入功能
- 支持在文章的第1-5段落后插入广告
- 完全支持 HTML 和 JavaScript 广告代码
- 如果文章段落数少于设置的段落数，广告会自动添加到最后一段
- 仅在单篇文章页面显示广告
- 简单易用的管理界面

## 安装方法

1. 下载插件压缩包
2. 登录到 WordPress 管理后台
3. 进入"插件">"安装插件"页面
4. 点击"上传插件"按钮
5. 选择下载的插件压缩包进行上传
6. 安装完成后，点击"启用插件"

## 使用说明

1. 在 WordPress 后台，进入"设置">"LeAddAds"
2. 勾选"启用插件"复选框以启用广告功能
3. 在"广告代码"文本框中粘贴你的广告代码（支持 HTML 和 JavaScript）
4. 从下拉菜单中选择要在第几段后插入广告（1-5段）
5. 点击"保存更改"按钮

## 注意事项

- 广告代码需要符合相关广告平台的规范
- 建议定期检查广告展示效果
- 如果文章段落数少于设置的段落数，广告将会显示在最后一段之后
- 广告仅会在单篇文章页面显示，不会在首页、分类页等其他页面显示

## 卸载说明

卸载插件时，所有插件相关的设置数据将被自动清除。
